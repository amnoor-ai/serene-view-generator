# 🌿 Serene View Generator

A simple web project that generates random nature images with matching titles.

Built to practice:
- Fetch API
- async / await
- DOM manipulation
- Git & GitHub workflow

---

## 🚀 Features

- Generates random nature images
- Displays a matching title
- Clean UI
- Button loading state

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Unsplash API

---

## 📸 How It Works

Click the button →  
Fetch data from Unsplash →  
Display image + title dynamically.

---

## 💻 Setup

1. Clone the repo  
2. Open `index.html` in your browser  
3. Click "Get Serene Images"
